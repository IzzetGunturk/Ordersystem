# 👨‍💻 Pizza order system

I programmed a pizza order system. The reason for this is to automate the work of a waiter. So the waiter is no longer necessary to ask what the customer want to eat. The customer can place an order by himself via the tablet. 

## How does the pizza order system works?

So the customers comes into the restaurant. They can order a pizza via the tablet. The chefs in the kitchen has their own user interface so where they can see all the orders and which table the order came from. When the order is done, one of the chefs can click on the button 'done'. After that the waiter brings the order to the customers.
